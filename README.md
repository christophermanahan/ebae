# Ebae

To start your Phoenix server:

  * Install Elixir 1.8.1
  * cd assets && npm install && cd ..
  * mix deps.get
  * mix ecto.setup
  * mix phx.server

Now you can visit [`localhost:4000`](http://localhost:4000) from your browser.

Default login information
  * username: username
  * password: password
